# Kunai Com Corrente

Chain.js goes into the website, then it can insert as many scripts as you like into it.  

## Getting Started

You must put the ```chain.js``` and the ```/kunais``` folder into your own server. Then, you can put a ```script src``` tag into the website you want to load.

## Built With

This project was builded with JS only.
